Плагин для автоматической отправки заявок из Contact Form 7 в мессенджер Max.ru через бота.

## 📋 Описание

CF7 to Max.ru Bot — это простой и эффективный плагин, который интегрирует популярный плагин форм Contact Form 7 с мессенджером Max.ru. Все заявки, отправленные через формы CF7, автоматически пересылаются в указанный канал/чат Max.ru через настроенного бота.

### 🚀 Возможности

- ✨ Автоматическая отправка заявок из Contact Form 7 в Max.ru
- 🔧 Простая настройка через админ-панель WordPress
- 📱 Поддержка Markdown форматирования сообщений
- 🧪 Встроенный тест отправки для проверки настроек
- 🎛️ Гибкое управление уведомлениями (вкл/выкл)
- 🔒 Безопасная работа с API через cURL
- 📊 Подробное логирование ошибок

## 📦 Установка

### Автоматическая установка

1. Скачайте ZIP-архив плагина
2. В админке WordPress перейдите: **Плагины → Добавить новый → Загрузить плагин**
3. Выберите скачанный архив и нажмите "Установить"
4. Активируйте плагин

### Ручная установка

1. Распакуйте архив в папку `/wp-content/plugins/cf7-to-maxru-bot/`
2. В админке WordPress перейдите в раздел **Плагины**
3. Найдите "CF7 to Max.ru Bot" и нажмите "Активировать"

## ⚙️ Настройка

1. После активации перейдите в **Настройки → CF7 to Max.ru Bot**
2. Заполните обязательные поля:
   - **Токен бота** — получите у администратора Max.ru или создайте нового бота
   - **Chat ID** — ID канала/чата (обычно отрицательное число)
3. Отметьте чекбокс **"Включить уведомления"**
4. Нажмите **"Сохранить изменения"**
5. Используйте кнопку **"Отправить тестовое сообщение"** для проверки работы

## 📝 Требования

- WordPress 6.0 или выше
- PHP 7.4 или выше
- Установленный и активированный плагин **Contact Form 7**
- Работающий бот в мессенджере Max.ru
- Включенное PHP расширение **cURL**
